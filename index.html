Modificado desde GitHub

Para clonar un repositorio desde Github a VSC, en el menu View, opción Command Palette escribir Git clone dar enter y pegar el link del repositorio de Github.


Si se hacen cambios en Github al proyecto, en VSC en la parte inferior izquierda (donde indica la rama) dar click en el botón refresh para actualizar los cambios en VSC de lo que se hizo en Github sirve como si se hiciera el proceso de push
